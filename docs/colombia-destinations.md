### Erläuterungen:
- **Leticia**: Nur per Flug ab Bogotá (~2h) → leeres Array
- **San Andrés / Providencia**: Nur per Flug vom Festland, untereinander per Flug (~20min) oder Boot
- **Caño Cristales**: Nur per Flug nach La Macarena → leeres Array
- **Nuquí / Bahía Solano**: Nur per Flug ab Medellín/Bogotá → leeres Array
- **Mompox**: Technisch per Bus ab Cartagena erreichbar (~8-10h), aber sehr umständlich
- **Ciudad Perdida**: Startpunkt ist Santa Marta (Trek geht von dort)

## 8. Seasonal Warnings

```javascript
// In gemini.js bei saisonalen Warnungen ergänzen:

// Kolumbien
if (country === 'colombia') {
  if (season === 'spring') {
    // Mar–Mai: Ende der Trockenzeit, Semana Santa
    seasonWarning = 'Frühling (Mär–Mai): Trockenzeit endet, ideal für die meisten Regionen. Semana Santa (Ostern) ist Hochsaison – Kolonialstädte wie Popayán und Mompox sehr voll. Buchung im Voraus empfohlen.';
  } else if (season === 'summer') {
    // Jun–Aug: Regenzeit in den Anden, trocken an Karibikküste
    seasonWarning = 'Sommer (Jun–Aug): Regenzeit in den Anden (Bogotá, Eje Cafetero), aber beste Zeit für Karibikküste. Caño Cristales öffnet ab Juni. Buckelwale an der Pazifikküste (Nuquí) ab Juni. Nachmittägliche Schauer in den Bergen sind kurz und heftig.';
  } else if (season === 'autumn') {
    // Sep–Nov: Regenzeit, Caño Cristales letzte Chance
    seasonWarning = 'Herbst (Sep–Nov): Regenreichste Monate in vielen Regionen. Letzte Chance für Caño Cristales (schließt Mitte November). Wale an der Pazifikküste bis Oktober. Weniger Touristen und günstigere Preise.';
  } else if (season === 'winter') {
    // Dez–Feb: Trockenzeit = beste Reisezeit
    seasonWarning = 'Winter (Dez–Feb): BESTE REISEZEIT – Trockenzeit in den meisten Regionen. Weihnachten und Silvester Hochsaison in Cartagena, San Andrés und an den Stränden. Bogotá kühler (12-18°C). Preise in Touristen-Hotspots deutlich höher.';
  }
}
```
